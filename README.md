# Portfolio

Personal portfolio site. Next.js 16, React 19, Tailwind CSS v4.

Design tokens live in `app/globals.css` (single source of truth) and are documented in [`docs/design-tokens.md`](./docs/design-tokens.md).

## Scripts

```bash
pnpm dev           # start dev server on http://localhost:3000
pnpm build         # production build
pnpm start         # serve the production build
pnpm lint          # ESLint
pnpm format        # Prettier — write
pnpm format:check  # Prettier — check only (CI)
```

## Requirements

- Node 20+
- pnpm
